# minimal (Preset)

A minimalistic starter preset for understanding Batonel without being overwhelmed by a large architectural pattern.

This preset gives you a functional baseline to experiment with Batonel's initialization, contract generation, and scaffolding commands.

## Architecture

- **Style:** `simple`
- **Ecosystem:** `generic`

## Defined roles

- `entity`: Represents a core business concept.
- `usecase`: Executes one application behavior.

## How to use

```bash
batonel init --preset minimal
batonel plan --mock
```
